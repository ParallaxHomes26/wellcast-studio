Build a Next.js 14 SaaS application called Wellcast Studio. Use the App Router, Tailwind CSS, and TypeScript.
Install these dependencies:

@supabase/supabase-js
@supabase/auth-helpers-nextjs
@stripe/stripe-js
stripe
lucide-react

Set the body background to #F3F2EE and default font to Inter.
Build these four foundational files:

src/lib/supabase.ts — Supabase client initialization using environment variables
src/lib/stripe.ts — Stripe client initialization using environment variables
src/lib/anthropic.ts — a single async function called generateAssets that accepts an episode brief object and a CTA string, makes a POST request to /api/generate, and returns the response
src/types/index.ts — TypeScript interfaces for:

EpisodeBrief (show_name, episode_type, host_name, guest_name, guest_credentials, health_niche, secondary_niches, core_topic, key_points, target_listener, tone, quotable_moments, actionable_takeaways, sensitive_topics, trust_signals, episode_length_signal)
EpisodeRun (id, user_id, created_at, episode_title, health_niche, episode_type, assets, confidence_score, credibility_flags, transformation_statement)
SubscriptionTier (free_trial, starter, pro, founding_member)
