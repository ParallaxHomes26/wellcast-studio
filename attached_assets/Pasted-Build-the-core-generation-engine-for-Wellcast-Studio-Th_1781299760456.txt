Build the core generation engine for Wellcast Studio. This is the most important part of the application.
1. Build the API route for extraction (src/app/api/extract/route.ts)
A POST endpoint that:

Receives a JSON body with fields: input_method (transcript/manual/url), transcript (string), manual_brief (object), url (string), and cta (string)
Calls the Anthropic API using ANTHROPIC_API_KEY from environment variables
Uses model claude-sonnet-4-6 and max_tokens 2000
Runs this extraction prompt:

You are the content intelligence engine for Wellcast Studio — a tool built exclusively for health and wellness podcasters.

Your job is to read a raw podcast transcript or episode brief and extract a clean, structured episode brief. This brief will be used as the foundation for generating all episode content assets.

Return ONLY a valid JSON object with this exact structure — no preamble, no markdown, just valid JSON:

{
  "show_name": "podcast name or null",
  "episode_type": "solo | guest_interview | co_hosted | panel",
  "host_name": "host name or null",
  "guest_name": "guest full name or null",
  "guest_credentials": "guest title and expertise in 1 sentence or null",
  "health_niche": "primary niche",
  "secondary_niches": ["array of additional niches"],
  "core_topic": "single clearest 1-sentence description of episode",
  "key_points": ["5-6 specific key points"],
  "target_listener": "1 sentence describing exactly who this episode is for",
  "tone": "clinical_and_educational | warm_and_conversational | story_driven | practical_and_tactical | mix",
  "quotable_moments": ["3 most shareable quotes or paraphrases"],
  "actionable_takeaways": ["3 specific things listener can do after this episode"],
  "sensitive_topics": ["any medically sensitive topics needing careful handling"],
  "trust_signals": ["credentials, experience, or evidence mentioned"],
  "episode_length_signal": "short (under 20 min) | medium (20-40 min) | long (40+ min)"
}

EPISODE INPUT:
[insert episode content here]

Returns the parsed JSON brief
Returns a 400 error if input is missing
Returns a 500 error with message if Anthropic call fails

2. Build the API route for asset generation (src/app/api/generate/route.ts)
A POST endpoint that:

Receives a JSON body with: episode_brief (object), cta (string), user_id (string), run_id (string)
Verifies the user has an active subscription or trial in Supabase before proceeding — if not return 403 with message "Trial expired or no active subscription"
Runs ALL asset generation prompts in parallel using Promise.all()
Each prompt call uses model claude-sonnet-4-6, max_tokens 1500
Returns a single JSON object with all assets grouped by category
